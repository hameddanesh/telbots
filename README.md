# TelBots
A PHP Framework for developing Telegram Bots.
<br>
<br>

# Installation

install TelBots with composer:
<br>

```
composer create-project hameddanesh/telbots
```

or clone from github:
<br>

```
git clone https://github.com/hameddanesh/telbots.git
```
